BR Tag

Details:

 * The <br> tag is used to insert a line break in the text. It forces the text to move to the next line without starting a new paragraph.

Syntax of BR Tag

<p>This is the first line.<br>This is the second line.</p>

Example of BR Tag

        <p>
          Line 1

          Line 2

          Line 3
        </p>

📌 Key Points

* <br> stands for break.
* It is an empty tag, meaning it has no closing tag.
* Used inside paragraphs, headings, or any block of text where you want to break the line without extra spacing.

⚠️ Note

* Use <br> for small line breaks. Avoid overusing it for layout or spacing — use CSS for that instead.
