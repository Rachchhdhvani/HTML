DOCTYPE Declaration

Details:

 * It should be the very first line in your HTML document.
* It is not case-sensitive, but it's typically written in uppercase for convention:<!DOCTYPE html>.
* It tells the browser to render the page in standards-compliant mode using HTML5.

Syntax of DOCTYPE Declaration

<!DOCTYPE html>

Example of DOCTYPE Declaration

        <!DOCTYPE html>
            <html>
                <head>
                     <title>My Page</title>
                </head>
                <body>
                    <h1>Hello, world!</h1>
                </body>
            </html>

⚠️ Important Notes:

* The DOCTYPE declaration is not an HTML tag.
* It helps browsers render the page in standards mode.
* In HTML5, it's simplified to just <!DOCTYPE html> and does not require a URL or version number.
