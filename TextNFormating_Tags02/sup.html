SUP Tag

Details:

 * The <sup>tag is used display superscript text — that is, text raised slightly above the normal line.It’s commonly used for.
* Mathematical expressions (like exponents: x²).
* Footnotes (e.g., Note¹).
* Chemical formulas (e.g., H₂O → H2O, CO2).

Syntax of SUP Tag

<sup>Superscript text</sup>

Example of SUP Tag

        * html:
    <!DOCTYPE html>
        <html>
        <head>
            <title>SUP Tag Example</title>
        </head>
        <body>
            <p>This is x<sup>2</sup> + y<sup>2</sup> = z<sup>2</sup>.</p>
            <p>Water boils at 100<sup>o</sup>C.</p>
        </body>
        </html>

        *📝 Output:

            This is x² + y² = z²
            Water boils at 100°C

🧠 Additional Information

* Semantically neutral: <sup> is a visual tag, not a semantic one (screen readers read it normally).
* Pair tag: Always has both opening <sup> and closing </sup>.
* No line breaks: As an inline element, it stays within text lines.


👎 Avoid These Mistakes 

 
| Mistake                        | Better Approach                         |
| ------------------------------ | --------------------------------------- |
| Using <sup> for styling only   | Use CSS if you're just adjusting layout |
| Overusing for emphasis         | Use <strong> or <em> for that           |
| Leaving it unclosed            | Always close: <sup>2</sup>              |


🔹 Styling With CSS 

    html:
       <sup style="color: red; font-size: 15px;">E = mc<sup>2</sup></sup>
    Output:
        E = mc2
