MARK Tag

Details:

 * The <mark>tag is used to highlight or mark text, usually to indicate something important or relevant in a document.
* By default, it shows up with a yellow background, like a highlighter.

Syntax of MARK Tag

<mark>Highlighted text</mark>

Example of MARK Tag

        * html:
    <!DOCTYPE html>
        <html>
        <head>
            <title>MARK Tag Example</title>
        </head>
        <body>
            <p>This is a <mark>very important</mark> part of the sentence.</p>
        </body>
        </html>

        *📝 Output:
                This is a very important part of the sentence.

🖍️ Use Cases <mark>

* Highlighting search results.
* Emphasizing important parts of a message.
* Marking changes or attention areas in notes.

🟡 Accessibility Tip

* <mark> is semantic — it tells screen readers that the content is highlighted or noteworthy.

🔹 Styling the <mark> Tag with CSS

* You can customize the background and text color: 

html:
        <style>
            mark {
                background-color: lightgreen;
                color: black;
            }
        </style>
