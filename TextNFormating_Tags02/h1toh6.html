H1 to H6 Tag

Details:

 * In HTML, <h1> to <h6> tags are heading tags used to define headings of different levels. They are used to structure the content in a hierarchical manner.

Syntax of H1 to H6 Tag

        <h1>This is a Heading 1</h1>
        <h2>This is a Heading 2</h2>
        <h3>This is a Heading 3</h3>
        <h4>This is a Heading 4</h4>
        <h5>This is a Heading 5</h5>
        <h6>This is a Heading 6</h6>

Example of H1 to H6 Tag

        <h1> Main heading (most important) Largest </h1>
        <h2> Sub-heading, Smaller Then h1 </h2>
        <h3> Sub-sub-heading, Smaller than h2 </h3>
        <h4> Further level heading </h4>
        <h5> Less important heading </h5>
        <h6> Least important heading, Smallest </h6>

⚠️ Notes

* You should use only one <h1> per page for SEO and accessibility.
* Headings help search engines and screen readers understand page structure.
