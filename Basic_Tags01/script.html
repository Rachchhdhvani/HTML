SCRIPT Tag

Details:

 The <script> tag is used to add or link JavaScript to an HTML document. JavaScript is used to make the page interactive (e.g., form validation, animations, dynamic content). 

      | Location                  |      Purpose                                             |
      | --------------------      |  -----------------------------------------------------   |
      | Inside `<head>`     | Loads before page content (may delay rendering).         |
      | End of `<body>`     | Preferred - script runs after the page loads.            |
      | With `defer` or `async`   | Used in `<head>` to control loading behavior.      |

Syntax of SCRIPT Tag

Basic Syntax

<script>// JavaScript code here</script>>

External Syntax

<script src="script.js"></script>

Example of Internal JavaScript

    <!DOCTYPE html>
        <html>
        <head>
            <title>Internal Script Example</title>
            <script>
                 function showMessage() {
                   alert("Hello from JavaScript!");
                 }
            </script>
        </head>
        <body>
            <button onclick="showMessage()">Click Me</button>
        </body>
        </html>

Example of External JavaScript

        <!DOCTYPE html>
        <html>
        <head>
            <title>External Script Example</title>
            <script>src="main.js" </script>
        </head>
        <body>
            <button onclick="showMessage()">Click Me</button>
        </body>
        </html>

        JavaScriptfile:

        function showMessage() {
          alert("Hello from external JavaScript file!");
        }
