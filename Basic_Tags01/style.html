STYLE Tag

Details:

 The <style> tag is used to write internal CSS directly inside an HTML document. It is placed inside the <head> tag and allows you to style elements without using an external CSS file.

Syntax of STYLE Tag

<style> /* CSS rules go here */ </style>

Example of STYLE Tag

    <!DOCTYPE html>
        <html>
        <head>
            <title>Style Tag Example</title>
            <style>
                 body {
                     background-color: #f0f0f0;
                     font-family: Arial, sans-serif;
                }

                h1 {
                     color: blue;
                     text-align: center;
                }
            </style>
        </head>
        <body>
            <h1>Welcome to My Website</h1>
            <p>This is a paragraph styled using the <style> tag.</p>
        </body>
        </html>
