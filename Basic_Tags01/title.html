Titel Tag

Details:

 * It must be placed inside the <head> section of your HTML document.
* It contains plain text only (no HTML tags or formatting).

Syntax of TITLE Tag

<title>Your Page Title</title>

Example of TITEL Tag

        <!DOCTYPE html>
        <html>
            <head>
                <title>My First Webpage</title>
            </head>
            <body>
                <h1>Welcome to My Page</h1>
            </bod>
        </html>
