HEAD TAG

Details:

 * The <head> tag contains meta-information about the HTML document.
* It is placed inside the <html> tag and before the <body> tag.
* It does not display content directly on the web page.

Syntax of HEAD

<head> <!-- Metadata elements go here --> </head>

Example HEAD TAG

        <html>
         <head>
            <meta charset="UTF-8">
            <title>Page Title</title>
            <meta name="description" content="A short description of the page">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <script src="script.js"></script>
         </head>> 
        </html>
