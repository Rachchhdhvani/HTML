HTML TAG

Details:

 * <html> is the root element of an HTML document.
* All other elements like <head>, <body>, etc., are nested inside it.
* It usually includes a lang attribute to specify the language of the document.

Syntax of HTML

<html> ... </html>

Example HTML TAG

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <title>My Website</title>
            </head>
            <body>
                <h1>Welcome!</h1>
            </body>
        </html>
